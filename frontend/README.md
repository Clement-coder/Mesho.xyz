## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the app.

## Project Structure

- `/app` - Next.js app router pages and components
- `/components` - Reusable UI components
- `/lib` - Utilities and context providers
- `/public` - Static assets
